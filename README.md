# smart-auditor-org

## Estrutura de Diretórios

- **smart-config/**: configurações gerais e setup inicial
- **smart-core/**: processamento fim-a-fim dos documentos
- **smart-email/**: leitura, envio e resposta de e-mails
- **smart-verificacao/**: modelos de verificação e testes de IA
- **smart-documentos/**: arquivos PDF gerados e recebidos
- **smart-utils/**: utilitários diversos (futuros)

## Como usar

1. Configure o '.env' em **smart-config/** a partir do '.env.example'
